# Vibe Coding 入门宝典

> 《Vibe Coding 入门宝典：非程序员的AI开发指南》

一本面向非程序员的 AI 开发实战指南，带你开启 Vibe Coding 之旅，用自然语言创造软件。

在线阅读：https://dreign.github.io/VibecodingBeginnersGuide/

---

## 📚 关于本书

本书探讨如何借助大语言模型（LLM）实现"一人公司"（One-Person Company）的愿景，让单个开发者能够完成以往需要整个团队才能完成的工作。

### 核心价值

- **认知先行**：先建立正确的思维模式，再学习具体工具
- **实战导向**：通过真实对话片段展示优秀与糟糕的 Vibe 交互
- **原理不变**：工具会变，但核心原理永恒
