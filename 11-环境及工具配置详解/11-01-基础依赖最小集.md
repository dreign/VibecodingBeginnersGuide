## 11.1 基础依赖最小集

### 你需要装什么

在开始 Vibe Coding 之前，你需要安装几个基础工具。不多，三个就够了。这三样东西是现代软件开发的基础设施——不管你用不用 AI，它们都在几乎所有项目的技术栈中。

**Node.js。** 这是现代 Web 开发的核心运行时。很多 AI 编程工具（包括 Claude Code）也依赖它来运行。安装完成后，你会同时得到 `node`（运行 JavaScript）和 `npm`（包管理）两个命令。

Node.js 为什么这么重要？因为 Vibe Coding 生态中有大量工具是基于 JavaScript/TypeScript 构建的。Claude Code 本身就是 npm 包，很多 AI 生成的 Web 应用使用 Next.js 或 React，这些都需要 Node.js。简单说：装了 Node.js，你就能运行大多数 AI 编程工具和 AI 生成的 Web 项目。

安装方式（Windows）：
1. 打开 https://nodejs.org
2. 下载左侧的 **LTS 版本**（Long Term Support，稳定版，不要下载右侧的 Current 版本）
3. 双击安装包，一路点击"Next"
4. 安装完成后，**重启终端**（重要——不重启终端可能找不到 node 命令）
5. 在终端中输入 `node --version`，如果显示 `v18.x` 或更高版本号，说明安装成功

安装方式（macOS）：
- 同样访问 https://nodejs.org 下载 LTS 版本安装
- 或者用 Homebrew：`brew install node@22`
- 终端输入 `node --version` 验证

> **小提示：** 如果你需要在多个 Node.js 版本之间切换（比如一个项目需要 v18，另一个需要 v22），可以用 `nvm`（Node Version Manager）来管理。但刚起步时不需要——装好 LTS 版本先用起来。

**Python 3。** 虽然不是所有项目都需要，但在数据处理、脚本编写、AI 相关任务中很常用。很多 AI 库（如 LangChain、AutoGPT 的衍生工具）需要 Python 环境。如果你以后想自己写一些数据处理脚本，Python 也是最佳选择。

安装方式（Windows）：
1. 打开 https://python.org 下载最新 Python 3
2. **安装时务必勾选底部的"Add Python to PATH"**——这个选项让终端能找到 python 命令。如果不勾选，之后你可能会遇到"找不到 python 命令"的错误
3. 点击"Install Now"完成安装
4. 终端输入 `python --version` 验证

安装方式（macOS）：
- macOS 可能自带 Python 3，但版本可能比较旧
- 推荐用 Homebrew 安装最新版：`brew install python@3.13`
- 终端输入 `python3 --version` 验证（macOS 上通常用 python3 而不是 python）

> **Windows 特别说明：** 有时你输入 `python` 会打开微软应用商店——这不是真安装。正确的安装方式是用上方的安装包，或者在终端中运行 `winget install Python.Python.3.13`。

**Git。** 版本管理。即使你一个人开发，Git 也是"不会丢代码"的底线保障。它能让你回到任意一个历史版本，能让你在尝试大改动时放心地"开个分支随便试"。

安装方式：
1. 打开 https://git-scm.com 下载对应系统的安装包
2. Windows 用户：安装时大部分选项保持默认即可，有一处需要注意——在"Choosing the default editor"这一步，建议选择 VS Code（如果已安装），而不是默认的 Vim（Vim 的学习曲线比较陡）
3. 终端输入 `git --version` 验证

安装完成后，你还需要配置你的名字和邮箱（Git 提交时会用到）：
```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```
以后每次用 Git 提交修改，系统都会记录是谁提交的。这个配置只做一次就好了。

### 包管理器：npm / pip

安装完 Node.js 和 Python 后，你会有两个包管理器：

- **npm**：安装 JavaScript/TypeScript 库和工具。命令：`npm install 包名`
- **pip**：安装 Python 库。命令：`pip install 包名`

在 Vibe Coding 中，你很少需要手动运行这些命令——你告诉 AI "帮我装一下 express"，AI 会自动运行 npm install。但知道它们的存在是有帮助的——万一 AI 安装失败，你知道它在做什么。

补充一下 npm 的几个常用变体：
- `npm install -g 包名`：全局安装，安装后可以在任何目录中使用该命令
- `npm install --save-dev 包名`：安装为开发依赖，只有开发时需要
- `npx 命令名`：不安装直接运行某个包（临时使用）

### 安装中可能遇到的问题

**问题一：'node' 不是内部或外部命令。**

这表示 Node.js 没有正确安装，或者安装后没有重启终端。先重启终端试试。如果还不行，重新运行 Node.js 安装包，选择"Repair"（修复）。

**问题二：'python' 不是内部或外部命令，打开微软应用商店。**

Windows 上常见的问题。解决方式：
- 重新运行 Python 安装包，确认勾选了"Add Python to PATH"
- 或者打开"设置 → 应用 → 应用别名"，关闭"python.exe"和"python3.exe"的应用安装程序别名

**问题三：'git' 不是内部或外部命令。**

重启终端，如果还不行，检查 Git 是否真的安装了。可以在"开始"菜单中搜索"Git Bash"——如果能打开，说明 Git 已安装，只是没有加到 PATH 中。

### 验证清单

安装完成后，打开终端，逐一运行以下命令确认所有工具就绪：

```bash
node --version    # 应显示 v18.x 或更高
python --version  # 应显示 Python 3.x
git --version     # 应显示 git 2.x 或更高
npm --version     # 应显示 9.x 或更高
```

如果每个命令都输出了版本号，你的基础环境就准备好了。整个过程大约需要 10 分钟。

### 进阶准备（非必须但推荐）

如果之后你要开发的项目涉及数据库，以下工具会有用：

- **Docker Desktop**：用来在本地运行数据库（PostgreSQL、MySQL、Redis 等），不用自己在机器上装数据库软件。安装方式：https://docker.com 下载安装。
- **VS Code**：代码编辑器（第 11.3 节会展开）。安装方式：https://code.visualstudio.com 下载安装。

这两个不是起步必需——你可以先做第一个项目，当需要数据库时再装 Docker，需要编辑器时再装 VS Code。

### 本节要点

- 最小依赖集只有三个：Node.js（Web 开发的运行时，Claude Code 也依赖它）、Python 3（脚本和 AI 工具常用）、Git（版本管理/时间机器）。安装时间不超过 10 分钟。
- Windows 安装特别注意事项：Python 要勾选"Add Python to PATH"，Git 编辑器选 VS Code，安装完重启终端。
- 包管理器 npm（JS）和 pip（Python）可以让你安装几乎所有需要的库。Vibe Coding 中你很少需要手动操作它们。
- 常见安装问题：命令找不到 → 重启终端；Python 打开商店 → 关闭应用别名或重新安装；Git 报错 → 检查安装包。
- 安装完成后运行验证清单确认所有工具就绪。进阶推荐 Docker 和 VS Code，但不是起步必需。

### Vibe 练习

打开终端，运行：

> "请帮我生成一个脚本，检查当前系统是否安装了 Node.js、Python 3、Git 和 npm，并报告它们的版本号。如果某个工具缺失，提示用户如何安装。"

如果你已经完成了安装，做一个小测试：

> 在终端中创建一个新目录，初始化一个 Node.js 项目（`npm init -y`），然后让 AI "帮我在这个项目中安装 express"。观察 AI 如何自动运行 npm install。如果安装成功，说明你的 Node.js 和 npm 环境完全正常。
